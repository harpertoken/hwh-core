# hwh-core

A minimal height–width model with formal constraints (Lean 4).

## Quickstart

1. Install Lean via `elan`: https://lean-lang.org/lean4/doc/quickstart.html
2. Build:

   ```bash
   cd hwh
   lake build
   ```

3. Run the tiny demo executable:

   ```bash
   lake exe hwh_core
   ```

## What “HWH” means (here)

`HWH` is treated as a small core for reasoning about **H**eight/**W**idth and simple rectangle/layout invariants in Lean.
